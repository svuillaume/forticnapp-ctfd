# Bonus / hand-authored challenges

Drop hand-authored CTFd challenge JSON in here if you want to seed challenges
that don't come from the FortiCNAPP API (e.g. trivia about the Fortinet
Security Fabric, LQL syntax puzzles, an "interpret this raw event" challenge).

The current bridge only imports API-driven challenges. To import files here,
extend `forticnapp_ctf_api/bridge.py` with a second loader.
